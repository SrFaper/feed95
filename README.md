# feed95

feed95 es una aplicación de catálogo personal de videojuegos. Permite crear perfiles locales y gestionar una colección de juegos, registrando su estado, calificaciones y detalles de forma completamente offline.

## Características
* Sistema de múltiples perfiles locales.
* Añadir, editar y eliminar juegos de tu catálogo.
* Seguimiento del estado del juego (Pendiente, Jugando, Completado, Abandonado) y calificaciones.
* Adjuntar imágenes desde tu dispositivo o mediante URL.

## Descarga e Instalación
Puedes descargar la última versión para Android o Windows desde la pestaña de [Releases](https://github.com/SrFaper/feed95/releases).
* **Android:** Descarga el archivo `feed95.apk`, ábrelo y permite la instalación desde fuentes desconocidas si el sistema lo solicita.
* **Windows:** Descarga el archivo `feed95-windows.zip`, extrae su contenido en una carpeta y ejecuta el `.exe`.

# feed95

feed95 is a personal video game catalog application. It allows you to create local profiles and manage your game collection, tracking their status, ratings, and details completely offline.

## Features
* Local multi-profile system.
* Add, edit, and delete games from your catalog.
* Track game status (Pending, Playing, Completed, Abandoned) and ratings.
* Attach images from your device or via URL.

## Download and Installation
You can download the latest version for Android or Windows from the [Releases](https://github.com/SrFaper/feed95/releases) tab.
* **Android:** Download the `feed95.apk` file, open it, and allow installation from unknown sources if prompted by the system.
* **Windows:** Download the `feed95-windows.zip` file, extract its contents into a folder, and run the `.exe`.
